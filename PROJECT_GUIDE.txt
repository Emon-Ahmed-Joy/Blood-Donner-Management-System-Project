1. PROJECT OVERVIEW
-------------------
This is a Java Swing-based Desktop Application designed to connect 
blood donors with those in need. The system supports three distinct 
user roles: Administrators, Donors, and General Users.

Architecture:
- model: Core data structures (User, Donor, Admin).
- database: In-memory data management (to be replaced with MySQL).
- ui: Professional Swing interfaces with role-based navigation.

2. USER MANUAL
--------------

A. LOGIN SYSTEM
   - Two tabs: "User/Donor Login" and "Admin Login".
   - System automatically detects if a User is a Donor and redirects 
     them to the appropriate dashboard.

B. ADMIN DASHBOARD (Credentials: admin / admin123)
   - View total donor statistics (Available vs. Busy).
   - View all registered donor details.
   - Monitor active blood requests.

C. DONOR PROFILE (Credentials: emon@gmail.com / pass123)
   - View personal blood group and location details.
   - Check if anyone has requested their blood.
   - Use "Search for Blood" if they need blood themselves.

D. USER HOMEPAGE (Credentials: user@gmail.com / pass123)
   - For non-donors: Includes a "Register as a Donor" button to upgrade status.
   - Standard "Search for Blood" functionality.

E. SEARCH & REGISTRATION
   - Search by Blood Group, State, and Location.
   - Registration allows creating a standard account or a Donor account.
